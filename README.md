# Run Tests

`yarn run test` OR
`yarn run test -- index.test.ts`

# Run Locally

`yarn run dev`

# Run Server

`yarn run server`

// Call server with Curl
`curl -d '{"key1":"value1", "key2":"value2"}' -H "Content-Type: application/json" -X POST http://localhost:3000`

## Packages Included

- Jest
- Lodash
- Datastructures-js
- Express
- Nodemon (hot reload)
